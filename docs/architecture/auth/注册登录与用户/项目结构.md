### **模块示例：auth (认证授权模块)**

text

```
auth/
├── api/                  # 用户接口层 (面向外部调用)
│   ├── DTO/             # 数据传输对象：接收前端请求参数
│   └── Controller/      # 控制器：参数校验 + 调用应用层 + 统一响应
│
├── application/          # 应用层 (业务流程编排)
│   ├── Service接口/     # 服务接口定义
│   ├── Service实现/     # 业务流程编排（指挥者角色）
│   └── Convertor/       # DTO与Entity转换器
│
├── domain/               # 领域层 (业务核心)
│   ├── Entity/          # 充血模型：包含业务逻辑的实体
│   ├── Repository接口/  # 数据访问接口定义
│   └── Domain Event/    # 领域事件定义
│
└── infrastructure/      # 基础设施层 (技术实现)
    └── Repository实现/  # 数据访问具体实现（如MyBatis Mapper）
```



## 🔧 **各业务模块功能说明**

### **1. 通用基础设施模块**

- **`common/`** - 跨模块共享组件
  - `config/` - 系统配置类
  - `constant/` - 常量定义
  - `exception/` - 统一异常处理
  - `result/` - 统一响应格式
  - `util/` - 工具类

### **2. 核心业务模块**

#### **`auth/` - 认证授权模块**

text

```
功能：用户注册、登录、权限管理、部门管理
```
